## 🎨 Color System

| สี | Hex | ใช้สำหรับ |
|---|-----|----------|
| Primary | `#8B4513` (Coffee Brown) | Brand, buttons |
| Success | `#10B981` (Green) | Stock OK, success states |
| Warning | `#F59E0B` (Amber) | Near expiry, low stock |
| Danger | `#EF4444` (Red) | Expired, errors |
| Info | `#3B82F6` (Blue) | PREP type indicator |
| Neutral | `#6B7280` (Gray) | RAW type, secondary text |
| Background | `#FAFAF9` (Off-white) | App background |

- Border radius: `8px` (sm), `12px` (md), `16px` (lg)
